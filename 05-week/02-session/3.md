### Ejercicio 3

Se tienen las entidades **`bill`**, **`bill_item`** y **`product`**. Construya una **consulta SQL** que permita listar el nombre del producto y el valor total pagado por producto dentro de las facturas.

**Condiciones obligatorias:**

- Usar **`LEFT JOIN`**
- Alias requeridos:
  - `b` → `bill` | `bi` → `bill_item` | `p` → `product`
- Relaciones entre tablas:
  - `b.id = bi.bill_id`
  - **Aplicar `LEFT JOIN` sobre `p.id = bi.product_id`**
- Mostrar únicamente:
  - `p.name`
  - `bi.price * bi.count AS total_pay_product`

No se deben mostrar columnas adicionales ni modificar los alias definidos. 

---

### Salida esperada (ejemplo)

| name | total_pay_product |
|------|------------------|
| Laptop | 2000 |
| Mouse | 50 |
| Keyboard | 120 |
